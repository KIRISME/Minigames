# Auto-Shooter Dodger

Open `index.html` in a browser to play. The player automatically shoots at enemies; use arrow keys or WASD to move and dodge incoming projectiles.

Files:
- `index.html` – launcher
- `style.css` – simple styling
- `game.js` – game logic

How to run:

1. Open the folder in your file explorer.
2. Double-click `index.html` or serve the folder with a static server and open it in a browser.
